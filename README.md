# Smart-Ambulance-With-MQTT-Protocol
This project introduces the smart ambulance system utilizing mqtt protocol. This project mainly designed to enhance emergency medical services. Developed Health monitoring system for patient and Traffic management system for ambulance using RTOS. The integration of ESP32 microcontroller facilitates immediate health updates and GPS location tracking. The system employs the Message Queuing Telemetry Transport (MQTT) protocol to facilitate efficient communication between the ambulance and the hospital. In Traffic management system, when the ambulance is near the traffic light, the traffic light changes from red to green to avoid traffic.
